# linkman

I don't like linktree, so i made my own

## Installation

```sh
git clone https://github.com/nezutero/linkman.git
```

- To install all dependencies:

```sh
bun i
```

## Usage

- To run:

```sh
bun start
```

- To run in dev mode:

```sh
bun dev
```
